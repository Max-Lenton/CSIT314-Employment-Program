# CSIT314-Employment-Program
A Employment Program for CSIT314 Systems Development Methodologies

## What you need
- Python 3.9 +
- Flask
- SQLAlchemy
### Installation help
In your python linked terminal 
- `pip install Flask` 
- `pip install Flask-SQLAlchemy`
to check `flask --version, pip show sqlalchemy`

## How to run 
1. CD into the folder containing 'main.py'
2. `flask --app main run`
3. ctrl + C to exit
